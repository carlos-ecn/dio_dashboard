# dio_dashboard
Dashboard de Vendas
Elaborador: Carlos Eduardo Cruz Nakandakare

=====================================
1. Visão Geral do Projeto
=====================================
Este projeto tem como objetivo treinar em cima de uma base de vendas de clientes Microsoft da plataforma XBOX, onde contem dados
referente as assinaturas normal e de planos EA e Minecraft.

=====================================
2. Objetivo de Negócios
=====================================
Analisar o histórico de vendas dos planos durante o ano de 2024.

=====================================
3. Conteúdo de Dados do Dashboard
=====================================

Visão da venda de planos em valores e quantidade de clientes ao longo do ano de 2024.
As visão foram divididas de formma de dados completos, somente o AE Pass e somente o Minecraft Pass.
